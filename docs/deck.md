# Deck

Rules governing how a [deck](glossary.md#deck) is composed for all [party](glossary.md#party) members.

## Deck Limit

- Every [party](glossary.md#party) member has a [deck limit](glossary.md#deck-limit) — the maximum number of cards their [deck](glossary.md#deck) can hold
- The player's [deck limit](glossary.md#deck-limit) is fixed — TBD value
- Each [companion's](glossary.md#companion) [deck limit](glossary.md#deck-limit) varies — see [Companions](companions.md#decks--cards)

## Composition Rules

- No minimum [deck](glossary.md#deck) size
- Duplicate cards are allowed — the same card may appear multiple times in a [deck](glossary.md#deck)

## Acquiring Cards

- Cards are acquired as [loot](glossary.md#loot) rewards after [encounters](glossary.md#encounter)
- Additional acquisition methods (shops, events, quests) — TBD
