# Core Loop — Open Questions

> See [core-loop_spec.md](core-loop_spec.md) for current settled decisions.

- **Map structure** — single seamless open world or divided into distinct zones/regions?
- **Monster scaling trigger** — what causes enemies to get harder over time: distance travelled, time elapsed, encounters completed, or something else?
- **Combat action constraint** — is there an energy system, action points, or another mechanic limiting how many cards can be played per turn?
- **Hand draw behavior** — does the player draw a fresh hand each turn, or maintain a persistent hand between turns?
- **Final boss** — is there a fixed end goal or final boss, or is the run purely a "how far can you go" challenge?
- **Rest site placement** — are rest sites fixed locations on the map, or can the player save anywhere?
- **Meta progression specifics** — what exactly persists between runs (currency, unlocks, other)?
