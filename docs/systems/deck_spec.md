# Deck — Spec

> Structured reference extracted from [deck.md](deck.md). Open questions: [deck_tbd.md](deck_tbd.md).

## Facts

- Player deck limit: 20
- Companion deck limits: vary per companion ([TBD](deck_tbd.md) values)
- Minimum deck size: none
- Duplicate cards in the same deck: allowed

## Rules

- A deck belongs exclusively to one party member — not shared between members
- Deck contents can only be changed outside of combat (at rest)
- Cards are drawn from the deck during combat to form the hand

## Behaviors

- Transfer card from bag into deck (at rest)
- Return card from deck back to bag (at rest)
- Draw card from deck into hand (during combat)

## Relationships

- Each party member owns one deck
- Deck cards are sourced from the shared bag
- Deck feeds the hand during combat
