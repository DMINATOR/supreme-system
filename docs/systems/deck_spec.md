## Rules

- All acquired cards go into the shared bag first
- A card must be moved from the bag into a member's deck before it can be used in combat
- A deck belongs exclusively to one party member — not shared
- Cards cannot normally be moved between decks during combat

## Behaviors

- Add card to bag (on loot drop)
- Transfer card from bag to a member's deck
- Return card from a deck back to the bag
- Draw card from deck during combat

## Relationships

- One bag is shared by the entire party
- Each party member owns one deck
- Decks are assembled from cards held in the bag
