# Initiative

> Facts, rules, behaviors, and relationships: [initiative_spec.md](initiative_spec.md)

[Initiative](../glossary.md#initiative) determines the order in which participants act during combat. It is a value carried by cards — equipment cards set a participant's initiative before combat, and combat cards can modify it during a fight.

Participants act in descending initiative order, interleaved across both sides. When participants share the same initiative value, the tie is broken by a random roll to decide which side acts first among the tied group, then turns interleave by member position within that group (A1 → B1 → A2 → B2…).

If no initiative cards are equipped, all participants default to 0 and the tie-breaking rule applies to the whole combat.
